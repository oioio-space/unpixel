# Bundled fonts — attribution & licenses

UnPixel embeds the following redistributable typefaces so it can recover
redactions zero-config (by sweeping them; see `unpixel.RecoverMultiFont`). Each
is unmodified and shipped under its original license. Full license texts are in
[`licenses/`](licenses): SIL Open Font License 1.1 (`OFL-1.1.txt`) and the
Apache License 2.0 (`Apache-2.0.txt`).

| Liberation Serif | `LiberationSerif-Regular.ttf` | Times New Roman | © 2012 Red Hat, Inc., Reserved Font Name "Liberation" | OFL-1.1 |

The SIL OFL permits bundling and redistribution provided the license and
copyright notices accompany the fonts (this file and `licenses/`), the fonts are
not sold by themselves, and the Reserved Font Names are not used on modified
versions. UnPixel ships the fonts unmodified. Caladea is under the Apache
License 2.0 (no reserved-name restriction).
